# simple-notepad-java
A simple Java Notepad-style desktop application demonstrating GUI development and basic file I/O.

## Overview
This project implements a **simple Notepad-style desktop application** in Java.  
It allows users to create, edit, and manage plain text files through a graphical user interface.

The application was built to practice **Java GUI programming**, event handling, and file input/output.

## Technologies Used
- **Language:** Java
- **UI:** Swing / AWT
- **Concepts:** Event handling, desktop GUI development, file I/O

## Features
- Text editing area
- Basic menu actions (new, open, save, exit)
- File loading and saving
- Simple, user-friendly interface

## Project Structure
- `Main.java` – Application entry point
- `NotePad.java` – GUI layout and event handling logic

## How It Works
1. The application initializes a window with a text editing area
2. Menu actions trigger file operations and UI updates
3. User input is handled through event listeners
4. Text content can be saved to or loaded from files

## How to Run
```bash
javac *.java
java Main
```
